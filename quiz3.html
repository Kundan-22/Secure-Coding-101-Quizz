What is the primary purpose of implementing rate limiting?

Rate limiting helps prevent automated attacks like brute force by limiting the number of requests a user can make in a given period.

Which of the following helps mitigate the risk of using vulnerable third-party libraries?

Regularly updating and patching third-party libraries ensures that known vulnerabilities are fixed, reducing security risks.

Which of the following techniques helps protect against Cross-Site Scripting (XSS)?

Encoding user inputs ensures that any potentially malicious code is treated as text, preventing XSS attacks.

How can you ensure secure communication between microservices?

Mutual TLS (mTLS) provides both authentication and encryption, ensuring secure communication between microservices.

Identify the secure code snippet for protecting against XSS:

// Option A
document.getElementById('output').innerHTML = userInput;
// Option B
document.getElementById('output').textContent = userInput;

Using textContent ensures that user input is treated as text, preventing the execution of any embedded scripts and mitigating XSS attacks.

Which of the following is a secure practice for handling session IDs?

Storing session IDs in cookies with the HttpOnly and Secure flags helps protect them from being accessed through JavaScript or transmitted over insecure connections.

Identify the code snippet that properly validates user input to prevent SQL Injection:

// Option A
$input = $_POST['input'];
$query = "SELECT * FROM users WHERE username = '$input'";
// Option B
$input = $_POST['input'];
$query = "SELECT * FROM users WHERE username = ?";
$stmt = $db->prepare($query);
$stmt->execute([$input]);

Using parameterized queries with prepared statements ensures that user inputs are properly handled, preventing SQL Injection attacks.

What is the benefit of using a secure random number generator for cryptographic purposes?

A secure random number generator ensures that numbers are unpredictable, which is crucial for cryptographic security.

Which of the following headers helps mitigate clickjacking attacks?

The X-Frame-Options header prevents the web page from being embedded in iframes, mitigating clickjacking attacks.

Identify the secure code snippet for handling passwords:

// Option A
import hashlib
hashlib.sha256(password.encode('utf-8')).hexdigest()
// Option B
import bcrypt
bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt())

Using bcrypt provides a secure hashing mechanism, whereas sha256 alone is not sufficient for secure password storage due to its speed and lack of salting.

Which practice helps in ensuring secure code review?

Using automated tools to complement manual reviews helps identify security issues that might be missed by human reviewers alone.

What is a common security risk of using weak cryptographic algorithms?

Weak cryptographic algorithms can be easily broken, compromising the security of the encrypted data.

What is the purpose of implementing Content Security Policy (CSP)?

CSP helps mitigate XSS attacks by specifying which sources are allowed to load content on the web page.

Which of the following practices helps in securing API endpoints?

Implementing proper authentication and authorization ensures that only authorized users can access API endpoints, protecting sensitive data and operations.
