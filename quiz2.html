Q.1 Which of the following is a characteristic of secure password storage?

Using a strong, one-way hashing algorithm ensures that passwords are not easily recoverable if the database is compromised.

Q.2 Which of the following practices helps prevent Broken Access Control?

Role-based access control ensures that users only have access to the resources necessary for their role.

Q.3 What is the purpose of the HTTP Strict Transport Security (HSTS) header?

HSTS ensures that the browser only interacts with the server over HTTPS, protecting against man-in-the-middle attacks.

Q.4 dentify the secure way to handle user authentication:

Using `password_verify` ensures that the password is compared against a securely hashed password, whereas a plain comparison is insecure.

Q.5 Which technique helps in preventing Insecure Direct Object References (IDOR)?

Server-side validation and authorization ensure that users can only access resources they are authorized to view.

Q.6 What is the best practice for handling sensitive data in transit?

SSL/TLS encryption ensures that data transmitted over the network is protected from interception and eavesdropping.

Q.7 Identify the secure way to implement password hashing:

Using `bcrypt` provides a secure hashing mechanism, whereas `md5` is considered weak and vulnerable to attacks.

Q.8 What is a common method to prevent Cross-Site Request Forgery (CSRF)?

CSRF tokens help ensure that the request is coming from an authenticated user and not a malicious site.

Q.9 Why should security headers like Content Security Policy (CSP) be used?

CSP helps mitigate XSS attacks by specifying which sources are allowed to load content on the web page.

Q.10 What is the benefit of using multi-factor authentication (MFA)?

MFA adds an additional layer of security by requiring multiple forms of verification before granting access.

Q.11 Identify the code snippet that follows secure coding practices for input validation:

Using a regular expression to validate input ensures that only allowed characters are processed, preventing many types of injection attacks.

Q.12 What is the main goal of logging and monitoring security events?

Logging and monitoring help detect and respond to security incidents as they occur, enabling timely action.

Q.13 Identify the secure code snippet for preventing Insecure Direct Object References (IDOR):

Using parameterized queries with prepared statements ensures that user inputs are properly handled, preventing IDOR vulnerabilities.

Q.14 What is the benefit of using secure coding practices?

Secure coding practices minimize the risk of introducing security vulnerabilities, enhancing the overall security of the application.

Q.15 Which of the following is a secure way to handle sensitive data in transit?

SSL/TLS encryption ensures that sensitive data is securely transmitted over the network, protecting it from interception.
